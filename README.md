# 🐱 猫なでゲーム

モグラたたきスタイルの可愛い猫なでゲームです！

## 🎮 ゲーム概要

10個の穴から次々に出てくる猫をなでてあげるゲームです。
ただし、猫以外のものが出てきたら要注意！間違ってなでるとゲームオーバーになります。

## 📋 ゲームルール

### 基本ルール
- **制限時間**: 各レベル30秒
- **レベル数**: 全7レベル
- **目標**: 30秒間生き残ればレベルクリア

### 操作方法
1. 「ゲーム開始」ボタンをクリックしてゲームスタート
2. 穴から出てくるアイコンをクリック
3. **猫のアイコン**が出たら素早くクリックしてなでる
4. **罠のアイコン**が出たらクリックせずに待つ

### アイコンの種類

#### ✅ なでてOK（猫）
- 😺 笑顔の猫
- 😸 ニコニコ猫
- 😻 ハート目の猫
- 🐱 猫の顔
- 😽 キス顔の猫
- 😹 笑い泣きの猫
- 🙀 驚いた猫
- 🐾 猫の足跡

#### ❌ なでたらNG（罠）
- 🦴 骨
- 🐶 犬

### ゲームオーバー条件
1. 猫が出ているのになでなかった場合
2. 罠（猫以外）をクリックした場合

### スコアシステム
- 猫をなでると得点が加算されます
- レベルが高いほど高得点（基本点 × レベル）
- 基本点: 10点

### レベル進行
- レベル1〜6: クリアすると次のレベルへ進む
- レベル7: クリアすると全レベルクリア！レベル1から再挑戦可能
- レベルが上がるほど、猫の出現速度が速くなり難易度が上昇

## 🚀 遊び方

1. `index.html` をブラウザで開く
2. 「ゲーム開始」ボタンをクリック
3. 猫だけをなでて、罠を避ける！
4. 30秒間生き残ればレベルクリア

## 💡 攻略のコツ

- **よく見る**: 猫か罠かを素早く判断しましょう
- **焦らない**: 罠が出たら無理にクリックせず待ちましょう
- **素早く反応**: 猫が出たらすぐになでないとゲームオーバーです
- **複数同時出現**: レベル4以降は複数の猫が同時に出現します
